# Проект для работы с публичными API

Простой консольный проект на Python для работы с публичными API. Программа позволяет делать GET-запросы к различным API и получать структурированные данные.

## Описание

Проект состоит из двух основных файлов:
- `http_client.py` - модуль для выполнения HTTP-запросов
- `main.py` - главный файл с консольным меню

## Требования

- Python 3.7 или выше
- Библиотека `requests`

## Установка

### 1. Клонируйте репозиторий или скачайте проект

### 2. Создайте виртуальное окружение (если еще не создано)

```bash
python -m venv venv
```

### 3. Активируйте виртуальное окружение

**В Windows (PowerShell):**
```powershell
.\venv\Scripts\Activate.ps1
```

**В Windows (Командная строка):**
```cmd
venv\Scripts\activate.bat
```

**В Linux/Mac:**
```bash
source venv/bin/activate
```

> **Примечание:** Если в PowerShell возникает ошибка выполнения скриптов, выполните:
> ```powershell
> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
> ```

### 4. Установите зависимости

```bash
pip install -r requirements.txt
```

## Запуск программы

После активации виртуального окружения запустите:

```bash
python main.py
```

## Использование

После запуска программы появится консольное меню с следующими пунктами:

### 1. Сделать GET-запрос по URL (показать сырой JSON)
- Позволяет отправить GET-запрос на любой URL
- Выводит полный JSON-ответ от сервера
- Полезно для тестирования любых API

**Пример использования:**
- Введите URL, например: `https://api.github.com/users/octocat`

### 2. Информация о стране (Rest Countries)
- Получает информацию о стране через [Rest Countries API](https://restcountries.com/)
- Выводит:
  - Название страны
  - Столицу
  - Население

**Пример использования:**
- Введите название страны: `Russia` или `Germany`

### 3. Случайная собака (Dog API)
- Получает ссылку на случайное изображение собаки через [Dog API](https://dog.ceo/dog-api/)
- Выводит прямую ссылку на картинку

### 0. Выход
- Завершает работу программы

## Функционал http_client.py

Модуль `http_client.py` содержит функцию `get()`, которая:

- Отправляет GET-запросы с использованием библиотеки `requests`
- Обрабатывает успешные ответы (статус 200) и возвращает JSON
- Обрабатывает ошибки HTTP (не 200) и выводит сообщение об ошибке
- Обрабатывает сетевые ошибки (`requests.exceptions.RequestException`)
- Поддерживает параметры запроса и таймаут

**Сигнатура функции:**
```python
def get(url, params=None, timeout=10)
```

## Обработка ошибок

Программа обрабатывает следующие типы ошибок:
- HTTP ошибки (статус коды не 200)
- Сетевые ошибки (нет подключения, таймаут и т.д.)
- Пустые вводы пользователя

При возникновении ошибки выводится понятное сообщение на русском языке.
